# Ahmed Body Aerodynamics

This project explores transient CFD simulation of flow over a 25° Ahmed body using OpenFOAM.

The work includes:
- Solver setup (PIMPLE + k-omega SST)
- Debugging numerical issues
- Transient simulation up to 5 seconds
- Force extraction and drag estimation
