# Ahmed Body Aerodynamics

## Objective
Predict drag coefficient and analyze wake structure.

## Physics
- External turbulent flow
- Reynolds number ~ 2.8e6
- k-omega SST model

## Methodology
- Domain: 5L upstream, 10L downstream
- y+ ≈ 30
- Boundary layer mesh

## Results
- Cd = 0.298
- Error vs literature: 3%
