# FUTURE WORK

## 1. Higher Resolution Study (Grid Independence)

### Improvement

- Run simulations with finer grids (e.g., 512, 1024)

### Objective

- Capture smaller vortex structures  
- Reduce numerical diffusion  

### Insight

Results should become independent of grid size  

---

## 2. Adaptive Mesh Refinement (AMR)

### Improvement

- Use Basilisk’s adaptive mesh feature  

### Objective

- Refine grid only near vortices  
- Reduce computational cost  

### Benefit

High accuracy with lower computation time
