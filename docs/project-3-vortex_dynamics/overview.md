# PROJECT 3: VORTEX DYNAMICS (KELVIN–HELMHOLTZ INSTABILITY)

---

# OVERVIEW

## What is this project?

This project simulates **vortex formation due to velocity differences in a fluid**, known as **Kelvin–Helmholtz instability**.

When two layers of fluid move at different velocities, the interface between them becomes unstable. Small disturbances grow over time and roll into **spiral vortices**.

---

## Why is this important?

This phenomenon appears in:

- Atmospheric flows (cloud patterns)
- Ocean currents
- Jet flows and wakes
- Combustion and mixing layers

Understanding this helps engineers analyze:

- Mixing efficiency  
- Flow instability  
- Turbulence generation  

---

## What is done in this project?

- A **shear layer** is created using a velocity profile  
- A small disturbance is introduced  
- The flow evolves naturally  
- Vortices form and interact

## Final Understanding

This project demonstrates how **velocity differences in a fluid can generate instabilities**, which grow into complex vortex structures over time.

It is a fundamental problem in fluid dynamics and is widely used to study **turbulence and mixing**.
