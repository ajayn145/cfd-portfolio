    PROJECT SUMMARY

    Overview

    • Simulated vortex formation due to velocity shear using Basilisk (C)
    • Modeled Kelvin–Helmholtz instability in a 2D incompressible flow
    • Generated vortex roll-up and nonlinear flow evolution
    • Visualized results using velocity and vorticity fields

    Key Physics

    • Shear layer instability
    • Vortex formation and evolution
    • Advection-dominated flow
    • Effect of viscosity on flow structures

    Key Results

    • Small perturbations grow into large vortex structures
    • Clear vortex roll-up observed due to shear instability
    • Lower viscosity leads to sharper and more defined vortices
    • Transition from linear instability to nonlinear vortex interaction

    My Contribution

    • Implemented shear layer and perturbation in Basilisk
    • Tuned parameters (viscosity, thickness, perturbation)
    • Generated and analyzed vortex dynamics videos
    • Interpreted physical behavior of instability and flow evolution

    Tools & Technologies

    • Basilisk CFD
    • C programming
    • ppm2mp4 (visualization)
    • Linux / terminal workflow

    Applications

    • Atmospheric and ocean flows
    • Jet and wake instabilities
    • Mixing layers in engineering systems
    • Foundation for turbulence modeling

    Questions This Project Answers

    • How do vortices form in fluid flows?
    • What causes shear layer instability?
    • How do small disturbances grow into large structures?
    • How does viscosity affect flow behavior?

    One-Line Summary

    This project demonstrates how velocity differences in a fluid generate instabilities that evolve into vortex structures through Kelvin–Helmholtz instability.
